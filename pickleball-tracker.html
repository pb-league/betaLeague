Setup steps
1. Go to sheets.google.com and create a new Sheet
2. Click Extensions → Apps Script
3. Delete all existing code, paste the script below
4. Click Save (disk icon)
5. Click Deploy → New deployment
6. Type: Web app · Execute as: Me · Access: Anyone
7. Click Deploy — authorize Gmail access when prompted
8. Copy the web app URL and paste it in Settings above
